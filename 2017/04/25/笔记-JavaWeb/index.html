Java Web 学习笔记

JDBC

JDBC接口

  1. 驱动管理

    • DriveManager

  2. 连接接口

    • Connection
    • DatabaseMetaData

  3. 语句对象接口

    • Statement
    • PrepareStatement
    • CallableStatement

  4. 结果集接口

    • ResultSet
    • ResultSetMetaData

JDBC使用步骤

  1. 导入JDBC的jar包
  2. 加载驱动
  3. 获取Connection对象
  4. 创建Statement语句对象
  5. 执行SQL语句
    • execute(DDL)常用于执行DDL,DCL
    • executeUpdate(DML)常用于执行DML语句
    • executeQuery(DQL)常用于之赐你个DQL语句
  6. 处理执行结果
    • execute(DDL)如果没有异常则成功, 返回值为boolean, true为有数据集, false没有数据集
    • executeUpdate(DML)如果没有异常则成功, 返回数字, 更新记录的行数
    • executeQuery(DQL)如果没有异常则成功, 返回ResultSet结果集, 二维查询结果
  7. 关闭数据连接!
    • 必须关闭, 否则数据库可能会瘫痪
